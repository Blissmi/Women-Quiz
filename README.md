# Quiz App (Prototype)

This is a small Vite + React + TypeScript starter containing the `QuizHero` component produced from the Figma export. It uses Tailwind for styling and Framer Motion for animations.

Quick start

1. Install dependencies

```bash
npm install
```

2. Start dev server

```bash
npm run dev
```

Notes

- The `QuizHero` component is in `src/components/QuizHero.tsx`.
- Replace `src/assets/logo.svg` with your exported Figma asset and update the import path if necessary.
- Tailwind is configured in `tailwind.config.cjs`.
